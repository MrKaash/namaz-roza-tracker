# 📱 Namaz & Roza Tracker

Ek Muslim app jo aapki qaza namaz aur roza ka 
poora hisaab rakhe - bilkul aasaan tarike se!

## ✨ Features
- Qaza namaz calculator (baligh date se aaj tak)
- Qaza roza calculator (Ramadan track)
- Daily 5 waqt namaz mark karna
- Daily roza track karna
- Progress charts aur streaks
- Google Drive backup (WhatsApp jaisa)
- 100% Offline kaam kare
- Dark/Light mode
